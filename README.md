💰 Finance Dashboard
A personal budget and expense tracker built in React. Track your income, expenses, and budgets with charts and visual breakdowns — all in the browser, no backend needed.
🖼 Preview
Code
✨ Features
📊 Dashboard — income/expense totals, savings rate, monthly bar charts, category pie chart
💳 Transactions — add, delete, and filter transactions by type, category, or month
🎯 Budget Tracker — set monthly budgets per category with live progress bars
🏷 9 expense categories — Housing, Food, Transport, Entertainment, Health, Shopping, Education, Utilities, Other
📅 Sample data included so it works out of the box
💾 All data lives in React state (no server needed)

📱 How to Use
Adding a Transaction
Go to the Transactions tab
Select type (income / expense), category, amount, and date
Click Add
Setting Budgets
Go to the Budget tab
Edit the amount for each category
Click Save Budgets
Filtering Transactions
Use the dropdowns on the Transactions page to filter by type or category.
